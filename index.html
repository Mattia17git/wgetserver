DropServer — GitHub Pages edition

Static page, no backend of your own. Drop a file, it gets committed straight to your GitHub repo via the GitHub API (from your browser), and you instantly get a real, wget-able URL.

Repository settings

How do I get a token?
GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → restrict "Repository access" to this one repo → under "Permissions" set Contents: Read and write. Nothing else is needed. The token never leaves your browser except to talk directly to api.github.com.
